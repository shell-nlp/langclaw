# Web Search Skill

## Purpose

Search the web for up-to-date information to answer user questions accurately.

## When to Use

- User asks about recent events, news, or current data
- User asks to "search", "look up", or "find" something online
- You are unsure whether your training data is current enough
- The question involves prices, availability, or real-time status

## Process

1. Identify the core search query from the user's request
2. Use the available search tool with a concise, focused query
3. Synthesise the results into a clear, direct answer
4. Cite sources when providing factual claims from search results

## Notes

- Prefer specific queries over vague ones for better results
- If results are insufficient, refine the query and search again
- Always distinguish between information from search results and your own knowledge
